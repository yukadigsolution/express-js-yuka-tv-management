API Documentation v1

Gunakan endpoint di bawah ini untuk menghubungkan aplikasi Client (Player) ke server Yuka TV.

Authentication

Semua request ke API v1 wajib menyertakan header berikut:

x-api-key: centauri

Catatan: Ganti http://localhost:3000 pada contoh di bawah dengan URL Vercel Anda saat sudah di-deploy.

POST /api/v1/join

Mendaftarkan perangkat baru ke dalam server agar tercatat di menu Device Management.

Contoh Request (cURL):

bash / cURL 
curl -X POST http://localhost:3000/api/v1/join \
  -H "x-api-key: centauri" \
  -H "Content-Type: application/json" \
  -d '{"device_name": "STB Android Ruang Tamu"}'

Contoh Request (PHP):

PHP (cURL) 
<?php
$ch = curl_init('http://localhost:3000/api/v1/join');
$payload = json_encode(['device_name' => 'STB Android Ruang Tamu']);

curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_POSTFIELDS, $payload);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'x-api-key: centauri',
    'Content-Type: application/json'
]);

$response = curl_exec($ch);
curl_close($ch);

echo $response;
GET /api/v1/playlist

Mengambil seluruh data channel dan film (VOD) yang ada di server dalam format JSON.

Contoh Request (cURL):

bash / cURL 
curl -X GET http://localhost:3000/api/v1/playlist \
  -H "x-api-key: centauri"

Contoh Request (PHP):

PHP (file_get_contents) 
<?php
$options = [
    'http' => [
        'method' => 'GET',
        'header' => 'x-api-key: centauri'
    ]
];

$context = stream_context_create($options);
$response = file_get_contents('http://localhost:3000/api/v1/playlist', false, $context);

echo $response;
